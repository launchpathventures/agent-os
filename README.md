# Agent OS

**The universal platform for human-agent collaboration.**

Agent OS helps non-technical people define, monitor, review, and improve agent-operated business processes — across any domain.

---

## Core Ideas

- **Process is the primitive** — not tasks, not agents. A process has inputs, steps, outputs, quality criteria, and feedback loops.
- **Agents serve processes** — the platform matches the right executor (AI, script, rules, human) to each step.
- **Trust is earned** — every process starts supervised. Agents earn autonomy through track record.
- **Agents check agents** — review patterns (maker-checker, adversarial, spec-testing) ensure quality before humans see output.
- **Self-improving** — the platform detects degradation, diagnoses root causes, and proposes improvements. Humans decide.
- **Two modes** — *Explore* (conversational discovery → process definition) and *Operate* (dashboards, review queues, metrics).

## Architecture

Six layers: Process → Agent → Harness → Awareness → Learning → Human.
